## Project setup
**Clone this repository**
```bash
$ git clone git@github.com:asherginting/search-filter.git
```

**Install**
```bash
$ cd search-filter
$ npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
